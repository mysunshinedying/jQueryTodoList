# 시간 추적 Todo List 📝⏱️
![녹화_2026_01_20_10_11_13_463](https://github.com/user-attachments/assets/32af532c-16e7-4ed8-a529-29d0dee1682a)
jQuery로 만든 작업 시간 기록 기능이 포함된 웹 Todo List입니다.

## 주요 기능

- ✅ Todo 추가/삭제
- ✅ 완료 처리 (완료된 항목은 자동으로 아래로 정렬)
- ⏱️ 작업 시간 타이머 (시작/일시정지/중지/리셋)
- 📊 통계 기능 (완료/전체 개수, 총 작업 시간)
- 🔍 필터 기능 (제목/기한으로 검색)
- 💾 localStorage 자동 저장

## 기술 스택

- HTML5, CSS3, JavaScript
- jQuery 3.7.1
- Font Awesome 7.0.1

## 사용 방법

1. **Todo 추가**: 제목 입력 후 ➕ 버튼 또는 Enter
2. **완료 처리**: 체크박스 클릭
3. **타이머**: ▶️ 시작, ⏸️ 일시정지, ⏹️ 중지, 🔄 리셋
4. **통계**: "통계보기" 버튼으로 완료 개수 및 총 작업 시간 확인
5. **필터**: 제목/기한으로 검색

## 업데이트 예정

- **뽀모도로 타이머**: 30분 작업 타이머 기능 추가 (총 작업시간에 포함, 알람 기능 고려 중)
- **랩타임 기능**: 스톱워치 중간 시간 기록 기능 (개별 시간 추적)
- **색상 테마**: 블루, 옐로 등 다양한 색상 테마 지원 (현재 기본: 핑크)
